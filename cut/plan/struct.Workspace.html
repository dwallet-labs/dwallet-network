fn read<P: AsRef<Path>>(root: P) -> Result<Self>

Read members and exclude from the workspace section of the Cargo.toml file in directory root. Fails if there isn’t a manifest, it doesn’t contain a workspace section, or the relevant fields are not formatted as expected.
